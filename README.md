# VJ Project & IT Solution

A responsive, single-page website for VJ Project & IT Solution, offering student
project development and laptop/IT repair services in Gudiyatham, Vellore, Tamil Nadu.

## Features
- Hash-based client-side routing (Home / Services / About / Contact) — no page reloads
- Student project request form with category, tech stack, deadline, budget, and
  documentation options
- Contact/service request form with conditional laptop-repair fields
  (brand, model, issue type)
- Dual submission: Netlify Forms (with spam honeypot) + instant WhatsApp handoff
- Fully responsive, mobile-first design with accessible focus states
- No build step — pure HTML, CSS, and vanilla JavaScript

## Tech Stack
- HTML5, CSS3 (custom properties, grid/flexbox)
- Vanilla JavaScript
- Google Fonts (Plus Jakarta Sans, Inter, Playfair Display)
- Netlify Forms for backend-free form handling
